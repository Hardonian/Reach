# Fast Path Preset

Minimal decision path for quick, low-risk decisions.

## What's Included

- Single evidence requirement
- No signature requirements
- 1-day expiration
- Automated approval for low-risk

## Usage

```bash
# Preview
./reach presets apply fast-path --dry-run

# Apply
./reach presets apply fast-path --yes
```

## When to Use

- Low-risk operational decisions
- Routine maintenance approvals
- Automated tool runs

## When NOT to Use

- Security-sensitive changes
- Production deployments
- Financial decisions
