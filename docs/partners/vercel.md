# Vercel Integration

## Overview

Deploy Reach Runners as serverless functions on Vercel for lightweight, event-driven execution.

## Setup

1. Install the Reach SDK: `npm install @reach/sdk`
2. Create an API route in `app/api/reach/route.ts`.
3. Bind the Vercel AI SDK to Reach streams.
