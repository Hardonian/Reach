## Overview

Recipes are simple, deterministic pipelines that make Reach easy to use. They provide a "fun entrypoint" for new users while maintaining production-grade determinism guarantees.

## User-Facing Terminology

- **Recipe**: A simple pipeline configuration
- **Run**: A single execution of a recipe
- **Checkpoint**: A saved state during execution
- **Rules**: Policy enforcement points
- **Findings**: Issues or observations discovered during a run
- **Proof**: Cryptographic evidence of execution integrity
- **Trust Score**: A 0-100 score indicating workspace
